Below you will find various examples of reproductible code that I came up with over the years - mostly functions and packages developped in R to help with ecological data analysis. Hopefully, this will be updated over time.

moonlit package

moonlit - R package I’ve been deveoping since 2018. It is a model to predict moonlight intensity on the ground for any given place and time, based on the position of the moon and a number of correction factors. It provides illumination measures that are much more biologically meaningful than moon phase.
This packge is available from a github repository.
